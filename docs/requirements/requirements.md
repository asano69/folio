# Requirements
 
Folioは、ナレッジベースです。そのため以下の機能が要求されます。
- Library / Collection / Bookを単位としてスキャン資料を自由に整理する機能
- 書籍やページには永続性のある静的なURLが与えられ、Wikiなどから自由に参照できる機能
- 各書籍には、BibTeXと互換性のある書誌情報の他、目次情報を保存できる機能
- 各画像ページには、タグ・メモ・マーカーなどの注釈情報を保存できる機能


Folioは、多くの本を並行して効率的に読むためのシステムです。そのため以下の機能が要求されます。
- 本を整理する機能
- 読書ステータスを管理する機能
- キーワード・ノートを管理する機能
